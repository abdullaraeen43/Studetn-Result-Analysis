# Studetn-Result-Analysis

A data analysis project that explores factors influencing student academic performance using Python. The notebook performs exploratory data analysis (EDA) on a rich dataset containing student demographics, family background, and exam scores across Math, Reading, and Writing.

# Tech Stack
Python
Numpy
Pandas
Matplotlib
Seaborn

# Analysis & Key Findings

1. Gender Distribution
2. Parental Education vs. Student Scores
3. Parental Marital Status vs. Student Scores
4. Ethnic Group Distribution
5. Math Score Distribution
